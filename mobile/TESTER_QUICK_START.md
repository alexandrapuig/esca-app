# Esca Mobile App - Beta Tester Quick Start Guide

Welcome to Esca Beta Testing!

## Getting Started

1. Download TestFlight (iOS) or accept Google Play Beta invite (Android)
2. Install Esca app
3. Sign up with your email address
4. Verify your email (check spam folder)
5. Log in

## What To Test (10 minutes)

1. Add an item: tap + Add Item, enter "Milk", select "Dairy", tap Save
2. View item: see it in the list with expiry date
3. Delete item: tap Delete button
4. Try Pantry tab: add an item there too
5. View Recipes tab (may be empty)
6. Check Profile: see your email and stats
7. Tap Logout: verify you return to login screen
8. Log back in: confirm credentials work

## If You Have A Camera (5 minutes)

1. Tap + Add Item
2. Tap "Scan Barcode"
3. Point camera at product barcode
4. Watch product auto-populate
5. Tap Save

## Test Offline (5 minutes)

1. Turn on airplane mode
2. Open Esca
3. Your items should still show
4. Turn off airplane mode
5. Pull down to refresh
6. Items should sync

## Report Issues

- TestFlight (iOS): Shake device -> tap Feedback
- Google Play (Android): Leave review in Play Store
- Email: beta@escawaste.com
- Include: what you were doing, what happened, device model
